# Yueli Resume Writer Skill

这是跃历的可分享入口页。  
如果你要把“跃历简历写作方法”交给另一个 AI / Agent 使用，优先看这里，再进入下面的正式技能文件。

## 这个 Skill 适合做什么

- 把零散经历整理成可投递简历
- 按目标岗位重组表达，而不是机械改写
- 保留真实事实，避免编造学校、公司、时间、奖项、数据
- 自动优先实习经历、项目经历、创业 / 校园经历
- 用 STAR 写法补足每条经历的结果和岗位关联
- 控制版面密度，支持中文 / 英文简历

## 推荐给 AI 的使用方式

1. 先读用户给的原始简历、经历素材、目标岗位和 JD。
2. 先列出可用经历，再让用户确认取舍。
3. 优先保留最相关的实习经历，其次是项目、创业、校园经历。
4. 每条 bullet 尽量包含背景、动作、结果和岗位适配。
5. 最后检查日期、顺序、版面、照片、PDF 导出是否一致。

## 核心规则

- 事实优先，不造假
- 实习优先，不乱改分类
- 先重组内容，再做排版
- 预览和导出保持一致
- 一页精简版和完整多页版都可以支持
